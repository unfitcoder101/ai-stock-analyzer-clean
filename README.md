# AI Stock Intelligence System 🚀

A full-stack AI-powered stock decision system that combines:

- Machine Learning (Logistic + Random Forest)
- Technical Indicators (RSI, Moving Averages)
- Confluence-based decision engine
- Backtesting system
- Watchlist feature
- Multi-stock scanner

## Tech Stack
- FastAPI
- React
- pandas, sklearn
- yfinance

## Features
- Predict BUY/SELL with confidence
- Trade planning (entry, stop-loss, target)
- Insight generation
- Stock scanning
- Watchlist
